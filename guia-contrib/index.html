    Guia de Contribuição
    Iniciativa Extra

    O que fazer antes de começar a contribuir?

      É de suma importância que, antes de começar a comtribuir, conheça bem o projeto e seus padrões. Por isso, é aconselhado que realize os seguintes passos:

    • Acesse o repositório do Projeto Curumim no GitHub;
    • Navegue pela GitHub Page do projeto e explore mais sobre seus fundamentos e propósitos;
    • Leia o Código de Conduta do projeto;
    • Estude e entenda as Políticas do projeto.

    Como contribuir?

    Reportando um Bug

    • Cheque se não há algum Issue referente ao bug em questão;
    • Caso o bug não esteja contemplado em nenhum issue, abra um novo de acordo com o template para bug report;
    • Submeta um Pull Request seguindo os padrões e o fluxo de trabalho do projeto;
    • Seu Pull Request será revisado e agregará muito à aplicação!

    Alterando e/ou adicionando funcionalidades

    • Cheque se não há algum Issue referente a implementação em questão;
    • Caso tal implementação não esteja contemplada em nenhum issue, abra um novo de acordo com o template básico;
    • Submeta um Pull Request seguindo os padrões e o fluxo de trabalho do projeto;
    • Seu Pull Request será revisado e agregará muito à aplicação!

    Agradecemos a sua contribuição! 👊
